#### **实操步骤（请按顺序执行）：**

**步骤 1：创建项目文件夹**
在你的电脑上新建一个文件夹，命名为 `EasyTestPlatform`（这就是我们要做 7 天的项目根目录）。

**步骤 2：创建 Python 虚拟环境（非常重要）**
打开终端（CMD 或 PowerShell），进入该文件夹，执行以下命令隔离环境：

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
```
*(成功后，你的终端前面会出现 `(venv)` 字样)*

**步骤 3：安装依赖**
继续在终端中执行：

```bash
pip install flask
```

**步骤 4：编写核心代码**
在项目根目录下新建一个文件 `app.py`，复制以下代码并保存。这个文件模拟了一个后端服务，它有内存数据库、登录接口和查询接口。

```python
# app.py
from flask import Flask, request, jsonify
import sqlite3
import time

app = Flask(__name__)

# 初始化数据库函数
def init_db():
    conn = sqlite3.connect('test.db')
    c = conn.cursor()
    # 创建用户表
    c.execute('''CREATE TABLE IF NOT EXISTS users 
                 (id INTEGER PRIMARY KEY AUTOINCREMENT, 
                  username TEXT, 
                  password TEXT, 
                  balance REAL)''')
    # 插入一条默认测试数据
    try:
        c.execute("INSERT INTO users (username, password, balance) VALUES ('admin', '123456', 100.0)")
        conn.commit()
    except sqlite3.IntegrityError:
        pass # 数据已存在，忽略
    finally:
        conn.close()

@app.route('/api/login', methods=['POST'])
def login():
    """模拟登录接口"""
    data = request.json
    username = data.get('username')
    password = data.get('password')
    
    # 模拟一点业务耗时
    time.sleep(0.1) 
    
    conn = sqlite3.connect('test.db')
    c = conn.cursor()
    c.execute("SELECT id FROM users WHERE username=? AND password=?", (username, password))
    result = c.fetchone()
    conn.close()
    
    if result:
        return jsonify({"code": 0, "msg": "Login success", "data": {"user_id": result[0]}})
    else:
        return jsonify({"code": 1, "msg": "User or password error"}), 401

@app.route('/api/balance/<int:user_id>', methods=['GET'])
def get_balance(user_id):
    """模拟查询余额接口"""
    conn = sqlite3.connect('test.db')
    c = conn.cursor()
    c.execute("SELECT balance FROM users WHERE id=?", (user_id,))
    result = c.fetchone()
    conn.close()
    
    if result:
        return jsonify({"code": 0, "data": {"balance": result[0]}})
    else:
        return jsonify({"code": 1, "msg": "User not found"}), 404

if __name__ == '__main__':
    print("正在初始化数据库...")
    init_db()
    print("服务启动中，请访问 http://127.0.0.1:5000")
    # 启动服务
    app.run(debug=True, port=5000)
```

**步骤 5：启动服务**
回到终端，确保处于虚拟环境中，运行：

```bash
python app.py
```

**步骤 6：验证成果**
保持终端运行（不要关闭），打开浏览器或使用 Postman：
1.  访问：`http://127.0.0.1:5000/api/balance/1`
2.  如果你看到浏览器显示：`{"code": 0, "data": {"balance": 100.0}}`，说明服务搭建成功！
